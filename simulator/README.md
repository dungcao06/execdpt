# Simulator
JaxMARL-HFT / ABIDES environment wrappers and LOB simulation interface.

- `env.py` — Gymnasium-compatible wrapper around JaxMARL-HFT
- `lobster.py` — LOBSTER data loader and replay environment
- `almgren_chriss.py` — Almgren-Chriss propagator model (for synthetic LOB fallback)
