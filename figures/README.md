# Figures
Scripts to reproduce all paper figures and tables.

## Planned Figures
- `fig1_icl_curves.py` — In-context learning curves (Stock-OOD & Date-OOD panels)
- `fig2_baselines_bar.py` — Grouped bar chart across all baselines and splits
- `fig3_ablations.py` — 4-panel ablation grid (context, pool, quality, params)
- `fig4_trace.py` — Qualitative execution trace (price + fills + order sizes)
- `table1_stats.py` — Task pool statistics
- `table2_results.py` — Main results table with CIs and p-values
